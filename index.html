Linguagem de marcação Html

É considerada uma linguagem de marcação, pois é utilizada apenas para que o navegador interprete como deve ler seu site e transmitir para a tela.
E como seu navegador entende qual tipo de conteúdo ele deve mostrar?
Através das TAGS=servem para indicar ao navegador o tipo do conteúdo que ele deve mostrar, elas são representadas pelos sinais= < e />
exemplos de tags básicas= <!doctype html>, <head>, <body>
